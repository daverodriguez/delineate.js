Delineate.js demo

Maybe you've dealt with this problem before. You have a list of inline links with dividers in between. You don't want the first item on each line to have a divider, but you can't control where elements will wrap. CSS doesn't have a :first-in-line pseudoclass, so I created this plugin to help.

This list does not use delineate:

  • Aspernatur, quis!
  • Laudantium, sed?
  • Enim, quia!
  • Excepturi, fugit.
  • Nulla, veniam.
  • Consequatur, repellat.
  • Aperiam, labore!
  • Dignissimos, doloribus!
  • Animi, ullam?

If you're using ES6, you can import and use delineate.js as a module:


		import {delineate} from './delineate.js';

		document.addEventListener('DOMContentLoaded', function() {
			delineate('.delineate-me');
		});
